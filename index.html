Data Cleaning (Comprehensive Preprocessing)

Output file: MailigCruzGalopeTorrefrancaTamondong_CleanedDataset.csv
  1. Handle missing values: remove rows missing key fields (rentals and basic weather), then fill any remaining missing numeric values using a typical middle value (the median).
  2. Remove duplicates: drop repeated rows so each record is counted once.
  3. Handle extreme outliers: for total, casual, and registered rentals, unusually extreme values are limited (“capped”) to a reasonable boundary so they don’t dominate the results.
  4. Scale weather values if needed: temperature, “feels-like” temperature, humidity, and windspeed are only scaled if their values are not already in a 0–1 range.
  5. Fix data types: ensure time/category fields are stored as whole numbers and weather values as decimals.
  6. Rename columns for clarity: convert abbreviated column names into clearer names (e.g., hour/month/year, total rentals, humidity, weather situation).
  7. Add readable labels: create text labels for season (Spring/Summer/Fall/Winter) and weather (Clear, Mist/Cloudy, Light Rain/Snow, Heavy Rain/Snow).
  8. Validate ranges: check that hour is between 0–23 and month is between 1–12.
  9. Export the cleaned dataset: save the cleaned data to the output file shown above.

1. Descriptive Statistics 

def compute_basic_statistics(self):
    numeric_cols = ['total_rentals', 'casual_users', 'registered_users', 
                   'temp', 'feeling_temperature', 'humidity', 'windspeed']
    rows = []
    for col in numeric_cols:
        s = self.data[col].dropna()
        rows.append({
            'Variable': col, 
            'Count': int(s.count()), 
            'Mean': float(s.mean()), 
            'Median': float(s.median()), 
            'Std': float(s.std()), 
            'Min': float(s.min()), 
            'Max': float(s.max())
        })
    self.basic_stats = pd.DataFrame(rows)
    return self.basic_stats

2. Correlation Analysis 

def compute_correlations(self):
    # Calculate Pearson correlation coefficient
    self.correlation_matrix = self.data.corr(method='pearson')
    return self.correlation_matrix

def find_strongest_correlations(self, threshold=0.3):
    target = 'total_rentals'
    # Filter for correlations stronger than threshold (0.3)
    corr_series = self.correlation_matrix[target].drop(target).abs()
    strong_corr = corr_series[corr_series > threshold]
    return strong_corr.sort_values(ascending=False)

def train_linear_regression(self):
    self.lr_model = LinearRegression()
    self.lr_model.fit(self.X_train, self.y_train)
    
    y_pred_test = self.lr_model.predict(self.X_test)
    
    self.results['lr'] = {
        'test_r2': r2_score(self.y_test, y_pred_test),
        'mae': mean_absolute_error(self.y_test, y_pred_test),
        'rmse': np.sqrt(mean_squared_error(self.y_test, y_pred_test))
    }
    return self.lr_model

4. Decision Tree Training 

def train_decision_tree(self, max_depth=10):
    # Initialize with optimized depth
    self.dt_model = DecisionTreeRegressor(max_depth=max_depth, random_state=42)
    self.dt_model.fit(self.X_train, self.y_train)
    
    y_pred_test = self.dt_model.predict(self.X_test)
    
    # Store performance metrics
    self.results['dt'] = {
        'test_r2': r2_score(self.y_test, y_pred_test),
        'mae': mean_absolute_error(self.y_test, y_pred_test),
        'rmse': np.sqrt(mean_squared_error(self.y_test, y_pred_test))
    }
    return self.dt_model
